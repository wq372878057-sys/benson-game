## 产品简介

**禅定花园**是一款以佛教文化为核心主题的双端协同应用（iPhone + Apple Watch），通过拨动佛珠积累功德值，逐步将破败佛斋升华为西方极乐世界。本仓库为网页交互原型，完整还原了双端界面与核心交互逻辑。

---

## 核心功能

### 佛珠系统
- **三种珠串**：菩提珠（初始）、琉璃珠（1,000功德解锁）、红玛瑙珠（5,000功德解锁）
- **弹簧-阻尼物理引擎**：模拟真实念珠的惯性与阻力
- **透视缩放渲染**：俯视角视角，近大远小的立体效果
- **母珠卍字标记**：每18颗一组，母珠触发特殊音效

### 功德与境界系统
- **10级禅堂升级**：从破败佛斋到西方极乐世界
- **实时进度显示**：进度条 + 剩余功德提示
- **升级仪式动画**：金光爆发 + 粒子特效 + 偈语展示

### 音效系统（Web Audio API 程序化合成）
- **拨珠音效**：基于珠串材质的差异化音色（菩提/琉璃/玛瑙）
- **母珠音效**：叠加泛音，庄严感更强
- **升级音效**：三层叠加（低频钟声 + 高频泛音 + 短促钟鸣）
- **背景音乐**：随境界切换（晨钟 → 梵唱 → 莲池 → 天籁）

### 双端界面
| 功能 | iPhone | Apple Watch |
|------|--------|-------------|
| 佛珠拨动 | 滑动/点击 | 滑动/点击/Digital Crown |
| 禅堂场景 | 全屏沉浸式 | 进度环表盘 |
| 功德统计 | 详细图表 | 简洁数字 |
| 图鉴系统 | 10境界卡片 | - |
| 振动反馈 | - | 每100颗触发 |
| 设置 | 音效/振动 | 音效/振动 |

## 技术架构

```
zen-garden/
├── client/
│   ├── src/
│   │   ├── components/
│   │   │   ├── BeadCanvas.tsx      # 佛珠渲染引擎（Canvas + 物理）
│   │   │   ├── PhoneApp.tsx        # iPhone 端完整界面
│   │   │   └── WatchApp.tsx        # Apple Watch 端界面
│   │   ├── contexts/
│   │   │   └── ZenContext.tsx      # 全局状态管理
│   │   ├── lib/
│   │   │   ├── zenStore.ts         # 数据模型 + 本地存储
│   │   │   └── audioEngine.ts      # Web Audio 音效引擎
│   │   └── pages/
│   │       └── Home.tsx            # 主页面（双端模拟器）
│   └── index.html
└── README.md
```

### 技术栈
- **框架**：React 19 + TypeScript
- **样式**：Tailwind CSS 4 + 自定义 CSS 变量
- **动画**：Framer Motion
- **音效**：Web Audio API（程序化合成，无需外部文件）
- **渲染**：HTML5 Canvas（佛珠物理引擎）
- **存储**：localStorage（功德数据持久化）
- **字体**：Noto Serif SC + Cinzel Decorative + Noto Sans SC

## 操作说明

### 网页端
- **点击/滑动** 佛珠画布拨动佛珠
- **键盘**：空格键/→ 拨珠，← 反向拨珠
- **鼠标滚轮**（Watch区域）：模拟 Digital Crown

### 手机端（真机）
- **触摸滑动** 拨动佛珠
- **上滑** Watch界面进入统计
- **下滑** 进入设置

# 生存者传说 - 游戏设计文档

## 游戏概述
一款肉鸽生存风格的射击游戏。玩家选择关卡后，自动瞄准并射击不断出现的怪物，每波击败10个怪物后可选择技能升级，共10波关卡。

## 屏幕列表

### 1. 主页 (Home Screen)
- **内容**: 游戏标题、开始按钮、设置按钮
- **功能**: 点击"开始战斗"进入关卡选择界面

### 2. 关卡选择 (Level Select Screen)
- **内容**: 两个关卡卡片（关卡1、关卡2），每个卡片显示难度、怪物类型等信息
- **功能**: 点击关卡卡片进入战斗界面

### 3. 战斗界面 (Battle Screen)
- **内容**:
  - 顶部: 当前波次信息 (1/10)、生命值、得分
  - 中央: 游戏场景（主角在底部，怪物在屏幕各处）
  - 底部: 当前波次进度条
- **功能**:
  - 主角自动瞄准并射击怪物
  - 怪物实时生成、受伤、消灭
  - 当前波次所有怪物消灭后，进入技能选择界面

### 4. 技能选择界面 (Skill Selection Screen)
- **内容**: 3个可选技能卡片（每波随机）
- **功能**: 
  - 点击技能卡片选择该技能
  - 选择后返回战斗界面进入下一波

### 5. 通关成功界面 (Victory Screen)
- **内容**: 通关提示、总得分、返回主页按钮
- **功能**: 点击返回主页

### 6. 失败界面 (Game Over Screen)
- **内容**: 失败提示、最终得分、重新开始按钮
- **功能**: 点击重新开始返回关卡选择

## 主要用户流程

```
主页 
  → 点击"开始战斗"
    → 关卡选择
      → 选择关卡
        → 战斗界面 (第1波)
          → 击败10个怪物
            → 技能选择
              → 选择技能
                → 战斗界面 (第2-10波)
                  → ... (重复波次)
                    → 第10波完成
                      → 通关成功
                        → 返回主页
```

## 关卡设计

### 关卡1: 森林之战
- **背景**: 绿色森林场景
- **怪物类型**: 普通怪物（绿色小怪）
- **难度**: 简单
- **特点**: 怪物移动速度较慢，血量较低

### 关卡2: 火焰地狱
- **背景**: 红色岩浆场景
- **怪物类型**: 火焰怪物（红色小怪）
- **难度**: 困难
- **特点**: 怪物移动速度快，血量较高，攻击力强

## 技能系统

每波完成后，玩家可从3个随机技能中选择1个：

### 可选技能列表
1. **伤害提升** - 增加射击伤害 (+20%)
2. **射速提升** - 增加射击速度 (+30%)
3. **生命恢复** - 恢复部分生命值 (+30%)
4. **范围伤害** - 射击产生爆炸伤害
5. **穿透射击** - 子弹穿透多个怪物
6. **冰冻效果** - 射击冻结怪物减速

## 色彩方案

- **主色**: #FF6B35 (橙红色 - 主角、射击效果)
- **次色**: #004E89 (深蓝色 - UI按钮、背景)
- **强调色**: #F7B801 (金黄色 - 得分、成功提示)
- **背景色**: #1A1A1A (深灰色 - 主背景)
- **文字色**: #FFFFFF (白色 - 主文本)

## 游戏机制

### 战斗流程
1. 进入战斗界面，当前波次开始
2. 每秒生成1-2个怪物，最多同时存在10个
3. 主角自动瞄准最近的怪物并射击
4. 怪物被击中后显示受伤动画并减少血量
5. 怪物血量为0时消失，增加得分
6. 所有10个怪物消灭后，波次完成
7. 显示技能选择界面，玩家选择1个技能
8. 进入下一波（最多10波）
9. 第10波完成后显示通关成功界面

### 主角属性
- 初始生命值: 100
- 初始伤害: 10
- 初始射速: 1次/秒
- 位置: 屏幕底部中央

### 怪物属性
- 初始血量: 20 (关卡1) / 30 (关卡2)
- 移动速度: 1 (关卡1) / 1.5 (关卡2)
- 攻击力: 5 (关卡1) / 10 (关卡2)
- 生成位置: 屏幕上半部分随机位置

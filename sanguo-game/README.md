# 三国演义：乱世英杰录

> 策略驱动历史 RPG · iPhone + Apple Watch 双端协同 · 只卖外观，不卖数值

## 产品概述

《三国演义：乱世英杰录》是一款将真实健康数据与三国历史叙事深度融合的策略 RPG 游戏。玩家通过日常行走、心率监测、睡眠追踪等真实健康行为，驱动游戏内的国力成长与战役推进。

## 技术栈

- **前端框架**：React 19 + TypeScript + Vite
- **样式系统**：Tailwind CSS 4 + 自定义水墨主题
- **动画**：Framer Motion
- **数据可视化**：Recharts
- **状态管理**：React Context + useReducer
- **路由**：Wouter

## 游戏机制

### 国力计算公式

```
日国力 = min(步数/100, 100) + 睡眠奖励 + 连续奖励 - 疲劳惩罚
```

- 步数：每 100 步 = 1 国力，上限 100
- 睡眠：≥7 小时 +30 国力「休养生息」
- 连续：连续 3 天 +10，连续 7 天 +20
- 疲劳：>80 时 -20，>50 时 -10

### 心率指挥倍率

| 心率区间 | 状态 | 指挥倍率 |
|---------|------|---------|
| <60 bpm | 静息 | ×0.8 |
| 60–100 | 热身 | ×1.0 |
| 100–130 | 有氧 | ×1.1 |
| 130–150 | 无氧 | ×1.3 |
| >150 bpm | 临危不乱 | ×1.5 |

### 声望等级

布衣 → 里长 → 县令 → 郡守 → 刺史 → 将军 → 丞相 → 王侯 → 皇帝
